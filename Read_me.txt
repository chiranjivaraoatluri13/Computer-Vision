# Smart Cane AI Project

This project uses the YOLOv8 object detection system to identify objects from images and webcam video, designed as part of a smart cane assistive technology prototype.

---

## **Getting Started**

### **1. Clone or Download the Project**

- Place all scripts (like `detect_sample.py`, `detect_webcam.py`) in a folder on your computer.

### **2. Setup Python Environment**

- Recommended: Use [Anaconda](https://www.anaconda.com/) for easy package management.
- Python 3.8 or later is recommended.

### **3. Install Required Packages**

Open your command prompt in your project folder and run:

python detect_webcam.py
